# House Price Prediction

## Objective
Predict house prices using machine learning based on housing features.

## Dataset
California Housing Dataset (from sklearn)

## Features Used
- MedInc
- HouseAge
- AveRooms
- AveBedrms
- Population
- AveOccup
- Latitude
- Longitude

## Model Used
- Linear Regression

## Workflow
1. Data loading
2. Exploratory Data Analysis (EDA)
3. Feature selection
4. Train-test split
5. Model training
6. Prediction
7. Evaluation (MAE, RMSE)

## Results
The model predicts house prices with reasonable accuracy using linear regression.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
