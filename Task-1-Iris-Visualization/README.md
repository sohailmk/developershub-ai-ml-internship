# Iris Dataset Visualization

## Objective
This project explores and visualizes the Iris dataset using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Dataset loading and inspection
- Statistical analysis
- Scatter plots
- Histograms
- Box plots

## Dataset
Iris Dataset loaded using Seaborn.

## Key Insights
- Iris-setosa is clearly distinguishable.
- Petal features are strong indicators for species classification.
- Visualization improves understanding of data distributions and relationships.
