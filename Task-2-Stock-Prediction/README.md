# Stock Price Prediction (Machine Learning)

## Objective
Predict future stock closing prices using historical market data.

## Dataset
Fetched using Yahoo Finance (yfinance) for Apple (AAPL).

## Features Used
- Open price
- High price
- Low price
- Volume

## Model Used
- Linear Regression

## Workflow
1. Data collection using yfinance
2. Feature selection
3. Train/test split
4. Model training
5. Prediction
6. Evaluation (MAE, RMSE)
7. Visualization

## Results
The model was able to approximate stock price trends based on historical data.

- pandas

- scikit-learn
