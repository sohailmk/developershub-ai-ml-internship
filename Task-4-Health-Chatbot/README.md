# General Health Query Chatbot

## Objective
Build a simple AI chatbot that answers general health-related questions using Gemini API with prompt engineering.

## Tools Used
- Python
- Requests library
- Google Gemini API

## Model Used
- gemini-2.5-flash

## Features
- AI-powered health Q&A chatbot
- Safe response filtering (no diagnosis)
- Prompt-engineered medical assistant behavior
- Interactive command-line chatbot

## How It Works
1. User enters a health-related question
2. Input is sent to Gemini API
3. AI generates a safe, general response
4. Response is displayed in terminal

## Example
User: What causes fever?  
Bot: Fever is usually caused by infection...

## Safety Note
This chatbot does not provide medical diagnosis or treatment advice.
