# Mental Health Support Chatbot

## 📌 Objective
To build an AI-powered chatbot that provides empathetic, supportive, and human-like responses for mental health-related inputs using a pretrained language model.

---

## 🧠 Project Description
This project demonstrates a simple mental health support chatbot using prompt engineering and a pretrained transformer model (DistilGPT2). The chatbot is designed to respond in a calm and supportive tone to user messages related to stress, anxiety, and emotional well-being.

---

## ⚙️ Approach
- Used a pretrained text generation model (DistilGPT2)
- Designed custom prompts to enforce empathetic behavior
- Generated conversational responses using HuggingFace Transformers pipeline

---

## 🛠️ Technologies Used
- Python
- HuggingFace Transformers
- PyTorch
- Jupyter Notebook

---

## 🤖 Model Used
- distilgpt2 (lightweight transformer model)

---

## 💬 Example Usage

**Input:**
I feel stressed and anxious about my exams

**Output:**
A supportive and empathetic AI-generated response

---

## 📊 Features
- Text-based conversational chatbot
- Emotionally supportive response generation
- Lightweight and fast model
- Prompt-engineered behavior control

---

## ⚠️ Limitations
- Not fine-tuned on mental health datasets
- Responses may not always be accurate or clinically appropriate
- Should not be used as a medical or psychological tool

---

## 🚀 Future Improvements
- Fine-tuning on EmpatheticDialogues dataset
- Adding safety filters for sensitive queries
- Deploying as a web app using Streamlit or Flask
- Improving response coherence with larger LLMs
